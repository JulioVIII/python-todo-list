# Python To-Do List

A simple to-do list application written in Python.

## Features

- Add tasks
- View tasks
- Save tasks to a file
- Load tasks from a file
- Input validation
- Console application

## Technologies

- Python 3

## How to Run

```bash
python todo_list.py
```

## Example

```text
--- TO-DO LIST ---
1. Add task
2. View tasks
3. Save tasks
4. Load tasks
5. Exit
```
